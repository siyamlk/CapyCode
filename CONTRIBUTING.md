# Contributing to CapyCode

Thank you for wanting to help build Capy! This project is designed to be
approachable for first-time open-source contributors.

## Setup

See the Getting Started section in [`README.md`](README.md).

## Ways to contribute

- **Art**: replace placeholder sprites (`src/assets/capy/`) with real
  pixel art per [`docs/ART_STYLE.md`](docs/ART_STYLE.md)
- **Personality**: add dialogue lines that match
  [`docs/CAPY_VOICE.md`](docs/CAPY_VOICE.md)
- **Providers**: implement a new `AIProvider` (Anthropic, Gemini, Ollama)
- **Features**: pick up a [good first issue](README.md#good-first-issues)

## Pull requests

- Keep PRs focused — one sprite, one provider, one feature at a time
- If you're touching Capy's dialogue, check it against `docs/CAPY_VOICE.md`
- Run `npm run build` locally before opening a PR

## Code of conduct

Be kind. Capy would want it that way.
